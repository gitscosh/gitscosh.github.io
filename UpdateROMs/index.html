How to Update Your MAME ROMset

Scosh Aug 14, 2020 Retro Gaming

How to update your MAME ROMset between point releases and get crazy upload ratio on your private torrent.

Things You’ll Need

  1. Verified ROMset downloaded from your private tracker torrent
  2. Torrent file for new point release from same private tracker
  3. DAT file (either DAT or XML format) for new release
  4. Latest version of ROMVault
  5. Favorite torrent client (recommend qBittorrent for any platform)

Steps

This guide assumes you’re starting with a full set of ROMVault verified ROMs from the previous point release (example: 0.222, and you want to easily upgrade to 0.223.) It also assumes you have ROMVault set up and know how to use it. Check the ROMVault web site for a tutorial, if you need some help.

  1. Load up the dat file and verify the current ROMset you have downloaded against the datfile for the NEW point release. The goal is to reuse as much of the current set as possible, so you have a minimum to download. I don’t have to tell you how important that is on ratio sites.
  2. After running fixes, you’ve got a set that’s going to be as close as you can get to the new set before downloading.
  3. Load up the torrent file for the new set, and point the download directory to your current set. Then force recheck on the torrent. You should get well over 90% completion.
  4. Once the recheck is done, simply start the torrent up, and download the differing files. You’ll have a new multi-gigabyte ROMset for very little download.

If you start with the non-merged set, you can easily build the merged and split sets, and have even more to share with the community.
